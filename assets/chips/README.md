# Poker chip image

Drop the **poker chip** image here as:

```
chip.png
```

- A square image also works; I'll crop it to a circle.
- ~256px+ square.

Note: the game currently draws 5 chip colors by denomination ($5/$25/$100/$500…).
If you give me **one** chip image, every chip uses that single design (denominations
will no longer be color-coded). If you'd rather keep per-denomination chips, send
several named `chip-5.png`, `chip-25.png`, `chip-100.png`, `chip-500.png` and I'll
map them. Tell me which you want.
