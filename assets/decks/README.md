# Card-back deck designs

Drop your **card-back images** here. Players pick a deck design on the setup
screen, and it's used for the backs of all face-down cards.

## How to add your designs

Name each file `deck-01`, `deck-02`, `deck-03`, … and use any of these formats:

```
.png  .jpg  .jpeg  .webp
```

So `deck-01.png`, `deck-02.jpg`, etc. You can upload as many as you like.

### Recommended

- **Portrait playing-card shape** — roughly a 5:7 ratio (e.g. 500×700). They're
  shown rounded at the corners; images get cropped to fit, so keep the design
  centered.
- Keep each reasonably small (under ~300 KB) so the table loads fast.

## Labels

Each design gets a name shown under it in the picker. Once you upload the
images, just tell me the labels you want — or I'll name them based on how each
one looks (like the character portraits). The labels live in `src/decks.js`.
